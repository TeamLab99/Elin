NPC 스크립트

1. 퀘스트 매니저에서 npc id에 맞는 퀘스트를 불러온다.
2. npc와 대화 가능 텍스트를 껏다 킬 수 있게 한다.
3. npc와 상호작용하면 npc에게 있는 퀘스트 id에 맞는 퀘스트만 진행한다.
4. isTalk라는 bool 변수를 선언한다.
- 플레이어가 quest 대화를 받으면? isTalk는 True
- 플레이어가 quest 조건을 만족하면? 만족한 text를 출력하고 isTalk는 False, 퀘스트 id는 증가
- 아니라면 불만족 text를 출력한다.
5.  json 파일에 isHunt라는 변수와 isCollect라는 변수를 선언해서 사냥 퀘스트와 모으는 퀘스트를 분리한다.
- isHunt=True, MonsterID = 1001; MonsterCount=10;
- isCollect = True, ItemID = 101; ItemCount=10;
-> .. 좀 더 생각해보자
6. NPC와 플레이어가 상호작용이 된 상태라면 텍스트가 다 떨어질때까지 bool 변수를 변경시켜 다음 텍스트가 나오도록 한다. 
- 텍스트가 담긴 배열의 크기보다 커지면 대화 종료를 실행한다.
